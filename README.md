# 🚀 Arbitrary Precision Calculator (APC)

## 📌 Description  
Arbitrary Precision Calculator (APC) is a C-based project that performs arithmetic operations on very large integers beyond the limits of standard data types like `int` and `long`. It uses doubly linked lists to store digits and process numbers efficiently.

---

## 🔥 Features  
- Supports very large integers  
- Addition, Subtraction, Multiplication, Division, Modulus  
- Doubly Linked List implementation  
- Handles positive and negative numbers  
- Command-line interface (CLI)  
- Input validation and error handling  
- Efficient carry and borrow handling  

---

## 📦 Modules  
- Main Module  
- Validation Module  
- String to List Conversion  
- Addition  
- Subtraction  
- Multiplication  
- Division  
- Modulus  
- Linked List Utilities  
- Output Display  

---

## ⚙️ Technologies Used  
- C Programming  
- Data Structures (Doubly Linked List)  
- Dynamic Memory Allocation  
- Pointer Manipulation
